#Mongo Examples
==============


The main reason for this is to connect multiple applications to a unified MongoDB. 

###RailsMongoid 
Is a sample application with Rails4 and the latest version of mongoid

###MongooseExample
Is also a sample application developed on express.js with mongoose. 

###MongoExample
Is also a sample application developed on express.js. The database connection is accomplished by creating new models (with the Model function defined in the lib directory).
